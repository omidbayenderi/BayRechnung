# Project Brief: BayRechnung

BayRechnung is a comprehensive multi-role management application designed for construction firms. It provides tools for invoicing, project management, financial tracking, and worker coordination.

## Core Goals
- **Multi-Role Support**: Tailored experiences for Employers/Admins, Managers, Finance/Accountants, and Site Leads/Workers.
- **Unified Management**: Manage the entire company and various construction sites from a single dashboard.
- **Financial Control**: Real-time tracking of income, expenses, turnover, and debts.
- **Workflow Efficiency**: Simplified reporting (daily reports, stock reports) and project progress tracking using Gantt/Kanban views.
- **Robust Persistence**: Seamless data synchronization between local state and Supabase cloud storage, supporting offline-first workflows.

## Key Features
- **User Management**: Comprehensive module for managing employees, roles, and site assignments.
- **Reports**: Centralized section for various report types (daily, financial, inventory).
- **Financial Dashboard**: High-level overview of firm-wide financial status.
- **Construction & Workflow**: Interactive views for site stages and progress tracking.
- **Messaging/Notifications**: Internal communication system for managers and workers.
